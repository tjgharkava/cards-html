# Nike Cards Project

This is a simple HTML & CSS project showcasing Nike shoes cards with responsive design and hover animations.

## Features
- Responsive card layout (3 cards on desktop, 1 card on mobile)
- Hover effects on cards and buttons
- Flexbox layout
